#This project will require you to use the Visual Studio Terminal:

1. On the top left corner, click on "Terminal"
2. Click on "New Terminal".
3. Make sure the terminal is in the "wrappers-chat-bot/backend" directory.

#Install the following dependencies using pip:

1. fastapi
2. uvicorn
3. scikit-learn
4. numpy

#To start the chatbot, run the following command in the terminal:

npm run start
